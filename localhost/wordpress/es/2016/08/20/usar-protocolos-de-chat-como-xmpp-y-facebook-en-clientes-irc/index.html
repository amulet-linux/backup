Usar protocolos de Chat como XMPP y Facebook en Clientes IRC

20 de August de 2016 admin Uncategorized

Para fomentar el uso de la consola y facilitarnos la vida hoy les enseñaré como pueden usar IRSSI que es un cliente IRC para muchos protocolos de chat.

A mi me gusta mucho usar un sandbox de GNU/Linux en mi Android para usar aplicaciones libres lo más que pueda, las aplicaciones de terminal son más seguras así que investigando lo que más me convenía en mi Debian Stable era usar IRSSI con BitlBee, BitlBee nos da la ventaja de no tener que instalar programas demás porque funciona como Gateway

1. Instalamos IRSSI

escojan un servidor de bitlbee que esté funcionando: https://www.bitlbee.org/main.php/servers.html , el primero que hallé sirve (im.bitlbee.org:6667)

2. Entremos a irsii escribiendo irsii en la terminal y un enter

3. Añadimos el server con el comando:

/server add -auto -network bitlbee -port 6667 im.bitlbee.org

4. Cambiamos a la ventana del canal de bitlbee con Alt + 2

5. Nos registramos con este comando eligiendo una contraseña

register CONTRASEÑA

6. Ahora añadimos nuestra cuenta

account add jabber bitlbee@jabber.org QuickStart

Donde jabber sería el protocolo o el servicio de chat, los disponibles son: msn, twitter, yahoo, oscar y jabber para XMPP. bitlbee@jabber.org sería nuestra cuenta de usuario y QuickStart nuestra contraseña

7. Ahora añade contactos c:

add 0 r2d2@example.com

Donde 0 puede ser un numero y por lo general comienza en 0, normalmente para indexar nuestro contactos  y r2d2@example.com es el nombre de usuario del contacto

8. Ahora ya puedes chattear

Supón que tu amigo se llama Tux, entonces para escribirle escribirias su nombre seguido de dos puntos:

Tux: Hola, ¿cómo estás?
